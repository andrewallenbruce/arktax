# Provider Types Reference

Provider Types Reference

## Usage

``` r
provider_types
```

## Format

### `provider_types`

A data frame with 305 rows and 5 columns:

- code:

  Provider Type Code (Full)

- type:

  Provider Type Code (Type)

- type_description:

  Type Description

- spec:

  Provider Type Code (Specialty)

- spec_description:

  Specialty Description
