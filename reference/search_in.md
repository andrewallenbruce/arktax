# Search in data frame column if search term is not `NULL`

Search in data frame column if search term is not `NULL`

## Usage

``` r
search_in(x, column, what)
```

## Arguments

- x:

  vector or `<data.frame>`

- column:

  `<chr>` string of column name

- what:

  to search for in `column`

## Value

vector or `<data.frame>`
