# Generate random strings

Generate random strings

## Usage

``` r
random_string(n, ln, p = "[A-Z0-9]")
```

## Arguments

- n:

  `<int>` number of strings to generate

- ln:

  `<int>` length of each string

- p:

  `<chr>` pattern to sample from; default is `"[A-Z0-9]"`

## Value

`<chr>` vector of random strings
